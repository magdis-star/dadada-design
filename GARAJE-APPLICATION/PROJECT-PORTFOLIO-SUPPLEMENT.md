## PROJECT 1: ELEMENTAL KIDS CLUB
**Educational Landing Page with Dual Conversion Funnels**

**Client**: Self-published author of children's activity books
**Industry**: Educational Publishing
**Timeline**: 3 weeks (brief to launch)
**Budget**: €650 | Delivered under budget
**URL**: elementalkidsclub.com

### Project Management Approach

**Discovery & Planning** (Week 1)
- Conducted client intake interview to understand dual business objectives: Amazon sales + email list building
- Performed competitor analysis of 5 similar educational products
- Created project charter with deliverables, timeline, and success metrics
- Defined two conversion funnels: direct-to-Amazon purchase + bonus lead magnet

**Execution** (Week 2)
- Applied Design Thinking methodology:
  - **Empathize**: Researched target audience (parents of 5-8 year olds concerned about screen time)
  - **Define**: Clarified problem statement and value proposition
  - **Ideate**: Designed wireframes with dual CTAs
  - **Prototype**: Built HTML/Tailwind CSS landing page
  - **Test**: Iterated based on client feedback (2 rounds)

- Managed vendor coordination:
  - Wild Mail integration for email marketing (double opt-in)
  - WordPress theme customization for client autonomy
  - GDPR compliance setup (cookie banner, privacy policy)

**Delivery & Handoff** (Week 3)
- Launched on elementalkidsclub.com
- Provided WordPress training for client (3 testimonials, 7 FAQs, URL editing)
- Created comprehensive documentation: "How to Edit Your Site" guide
- Post-launch support: 2 rounds of minor adjustments

### Key PM Deliverables
✅ Project plan with Gantt chart (Asana)
✅ Client communication log (weekly check-ins)
✅ Budget tracking (under €650 target)
✅ Risk register (GDPR compliance, Wild Mail API issues)
✅ Testing checklist (cross-browser, mobile responsive, form submission)
✅ Handoff documentation

### Results
- **On-time delivery**: 3 weeks from brief to launch
- **Under budget**: Client paid €650, actual cost €580
- **Client autonomy**: 100% - client can edit content without developer
- **Two functioning funnels**: Amazon purchases + email lead capture working simultaneously

**Client Testimonial**:
> "Magda entendió perfectamente que necesitaba dos cosas: vender libros en Amazon Y construir mi lista de email. La integración con Wild Mail funciona perfectamente, y lo mejor es que puedo editar testimoniales y FAQs yo misma sin llamarla cada vez."

## PROJECT 2: TIE THE CELTIC KNOT
**WordPress Redesign & SEO Strategy for Wedding Celebrant**

**Client**: Wedding celebrant specializing in Celtic ceremonies
**Industry**: Wedding Services
**Location**: Ontario, Canada
**Timeline**: 4 weeks
**Budget**: €850
**URL**: tiethecelticknot.ca

### Project Management Approach

**Discovery Phase** (Week 1)
- Inherited existing DIY WordPress site with poor SEO performance
- Conducted SEO audit: identified 0 Google ranking for target keywords
- Stakeholder interview to understand business goals:
  - Primary: Rank for "Celtic wedding celebrant Ontario"
  - Secondary: Convert website visitors into consultation bookings
- Created project roadmap with 3 sprints

**Sprint 1: UX/UI Redesign** (Week 2)
- Heuristic evaluation of existing site (usability issues identified)
- Competitor analysis of 3 successful celebrant sites
- Created new information architecture
- Designed wireframes in Figma (3 iterations with client)
- Selected Celtic-inspired design system (greens, gold, traditional motifs)

**Sprint 2: Development & Content** (Week 3)
- WordPress theme customization
- Implemented responsive design (mobile-first approach)
- Coordinated with copywriter for SEO-optimized content
- Added testimonial section (social proof)
- Created ceremony gallery (visual storytelling)

**Sprint 3: SEO & Launch** (Week 4)
- On-page SEO: meta descriptions, alt tags, header hierarchy
- Google My Business optimization
- Submitted sitemap to Google Search Console
- Set up analytics tracking (GA4)
- Conducted pre-launch QA testing

### Key PM Deliverables
✅ SEO audit report with recommendations
✅ Content strategy document (keyword mapping)
✅ Design mockups (Figma)
✅ Sprint planning with client approval gates
✅ Testing protocol (devices, browsers, load time)
✅ Analytics setup and initial baseline

### Results
- **Timeline**: Delivered in 4 weeks (on schedule)
- **SEO success**: Client ranking on Google page 1 for target keywords within 3 months
- **Conversion improvement**: Contact form submissions increased (client report)
- **Technical performance**: Page load time <2 seconds

**Client Feedback**:
> "My old site wasn't bringing me any clients. After the redesign, I started appearing in Google searches and getting consultation requests. The site finally looks as professional as my ceremonies."

## PROJECT 3: GONZALO MORALES ART GALLERY
**Minimalist Portfolio Site for Visual Artist**

**Client**: Professional visual artist
**Industry**: Fine Art
**Timeline**: 2 weeks
**Budget**: €500
**URL**: gonzalomorales.net (case study: dadadadesign.com/casos/gonzalo-morales-galeria-arte)

### Project Management Approach

**Brief & Planning** (Days 1-2)
- Client requirement: "Let the art speak" - minimal design, maximum focus on artwork
- Defined success criteria: Fast loading, easy navigation, mobile-optimized gallery
- Created mood board with minimalist design references (Saul Bass, Massimo Vignelli)

**Design & Development** (Days 3-10)
- Iterative design process:
  - **Lo-fi wireframes**: Gallery layout options (grid vs. masonry)
  - **Hi-fi mockups**: Color palette (black, white, single accent), typography (modern sans-serif)
  - **Interactive prototype**: Lightbox gallery with smooth transitions
- Developed custom WordPress theme
- Optimized images for web (lazy loading, WebP format)

**Testing & Launch** (Days 11-14)
- Cross-device testing (desktop, tablet, mobile)
- Load time optimization (<1.5s)
- SEO basics (artist name, artwork descriptions)
- Client training session (30 minutes: how to add new artwork)

### Key PM Deliverables
✅ Design brief with mood board
✅ Wireframes and mockups (Figma)
✅ Development timeline (2-week sprint)
✅ Image optimization guide for client
✅ CMS training video (screen recording)

### Results
- **On-time, on-budget**: 2 weeks, €500
- **Performance**: 1.3s load time, 95+ Lighthouse score
- **Client autonomy**: Can add new artwork independently
- **Design quality**: Clean, professional portfolio reflecting artist's aesthetic
